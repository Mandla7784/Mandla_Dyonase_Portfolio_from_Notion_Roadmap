# Mandla Dyonase — Portfolio

Web developer portfolio built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Sections (commit by part)

- **Part 1** — Project setup + Hero section ✅
- Part 2 — Navigation & layout
- Part 3 — Skills (with icons)
- Part 4 — Projects (Vercel)
- Part 5 — About & Contact
- Part 6 — Polish, meta, favicon
